# 示例数据说明

本文件夹用于存放 DataPilot 的内置示例数据。

## sales_sample.csv

销售示例数据，适合小微商家经营分析，字段包括：

- 日期
- 商品
- 销售额
- 订单数
- 城市

该数据用于测试：

- 字段识别
- 日期趋势分析
- 分类字段分析
- 异常值检测
- 相关性分析
- 图表推荐
- 清洗建议
- AI 报告生成

其中 2026-06-08 的销售额和订单数被设置为明显偏高，用于测试异常值检测能力。

## survey_sample.csv

问卷示例数据，字段包括：

- 提交时间
- 年龄段
- 城市
- 使用频率
- 满意度评分
- 推荐意愿
- 最需要改进的地方
- 开放反馈

该数据用于测试问卷数据分析场景，包括样本量判断、评分分析、选择题分布和开放反馈识别。

## expense_sample.csv

个人消费示例数据，字段包括：

- 日期
- 消费分类
- 商户
- 金额
- 支付方式
- 备注

该数据用于测试个人消费分析场景，包括总支出、消费结构、大额消费和时间节奏分析。
