# DataPilot：AI 数据分析工作台

DataPilot 是一个基于 Python、Streamlit 和 DeepSeek API 构建的 AI 数据分析工作台。

用户上传 CSV 或 Excel 文件后，系统可以自动完成数据概览、字段识别、基础统计、图表分析、异常值检测、相关性分析、数据清洗建议和 AI 分析报告生成。

当前版本：v2.0

---

## 一、当前版本定位

v2.0 是场景化分析基础版。

这一版本在 v1.0 公开测试能力之上，增加问卷数据分析、小微商家经营分析、个人消费分析和数据分析学习陪练四种场景。

---

## 二、当前支持的文件格式

支持：

- .csv
- .xlsx

建议：

- 表格第一行作为字段名
- 字段名尽量清晰，例如：日期、商品、销售额、订单数
- 尽量避免合并单元格
- 尽量避免多层表头
- 尽量避免空白标题列

暂不支持：

- .xls
- PDF
- Word
- 图片
- 数据库直连

## 三、当前核心功能

### 1. 数据概览

系统会自动识别：

- 行数
- 列数
- 重复行数
- 缺失值总数
- 缺失值比例
- 数值字段
- 日期字段
- 分类字段
- 文本字段

### 2. 数据详情

包括：

- 数据预览
- 字段质量检查
- 数值字段描述统计

### 3. 数据质量

包括：

- 异常值检测
- 数据清洗建议
- 高优先级问题提醒

### 4. 图表分析

包括：

- 自动生成图表
- 图表推荐
- 相关性分析

### 5. AI 分析报告

系统基于统计摘要调用 DeepSeek API，生成中文数据分析报告。

为降低 API 成本和隐私风险，AI 只读取统计摘要，不直接读取完整原始表格。

### 6. 场景化分析

包括：

- 通用表格分析
- 问卷数据分析
- 小微商家经营分析
- 个人消费分析
- 数据分析学习陪练

系统会根据当前场景生成对应的场景洞察，并影响 AI 报告的分析重点。

## 六、主要目录说明

### core/

存放具体的数据分析能力，例如文件读取、字段识别、基础统计、异常值检测、相关性分析、图表推荐和清洗建议。

### pipeline/

存放统一分析流程和 AI 报告上下文生成逻辑。

### sections/

存放页面展示模块，例如首页、上传区域、左侧边栏、工作台、数据概览、图表分析和 AI 报告区域。

### prompts/

存放 AI 提示词。

### utils/

存放 DeepSeek API 调用等通用工具。

## 七、注意事项

### 1. 不要上传 .env

.env 文件中保存 DeepSeek API Key，不能上传到 GitHub。

### 2. 不要上传 .venv

虚拟环境文件夹不需要上传。

### 3. 不要上传敏感数据

使用时不建议上传包含身份证号、手机号、银行卡号、真实客户名单等敏感信息的数据文件。

## 十、部署说明

更完整的部署步骤见 `DEPLOYMENT.md`。

### 本地配置

在项目根目录创建 `.env`：

```text
DEEPSEEK_API_KEY=你的 DeepSeek API Key
DEEPSEEK_MODEL=deepseek-chat
```

### Streamlit Cloud 配置

在 Streamlit Cloud 的 Secrets 中配置：

```toml
DEEPSEEK_API_KEY = "你的 DeepSeek API Key"
DEEPSEEK_MODEL = "deepseek-chat"
```

启动命令：

```bash
streamlit run app.py
```
